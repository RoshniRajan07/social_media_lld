Social Media Application:

This is a console-based Social Media Application developed in Java.
It allows users to sign up, connect with friends, create posts, and interact with posts such as liking them.

Features:
User signup with basic details

Add friends

Create posts

View posts (news feed)

Like posts

Functional Requirements:
Users can create accounts by providing name, age, location, and occupation.

Users can connect with other users as friends.

Users can create posts.

Friends can view posts in the news feed.

Users can like posts.
